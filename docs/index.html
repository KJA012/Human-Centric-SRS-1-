최종 추천 점수 공식:
Score = SRS × (1 + α × H)

기존의 기술적 우연성 점수(SRS)에 사용자의 능동적 선택권($\alpha$)과 창작자의 인간성 지표($H$)를 결합한 상생형 알고리즘입니다.
SRS (Serendipity Recommendation Score):
신선도(20%), 다양성(30%), 참신함(50%)을 수학적으로 합산한 베이스라인 점수입니다. 생성형 AI 콘텐츠와 인간 콘텐츠 모두에게 공평하게 적용되는 '기술적 재미'의 척도입니다.
H-Index (Human Authorship):
AI 탐지 모델과 크라우드 소싱 인증을 통해 산출된 '인간 창작 확률'입니다. 0에 가까울수록 AI 양산형, 1에 가까울수록 인간의 고유한 노고가 담긴 서사를 의미합니다.
Alpha ($\alpha$):
사용자가 안티-버블 슬라이더를 통해 직접 부여하는 가중치입니다. 최대 1.5까지 확장되며, 인간 창작물($H$)의 점수를 증폭시켜 필터 버블을 깨뜨리는 에너지가 됩니다.
