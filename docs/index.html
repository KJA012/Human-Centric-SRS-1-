🔍 알고리즘 상세 지표 설명서

F-Index
유저가 이미 시청한 뻔한 키워드를 필터링하고 장르적 시간 장벽을 계산하여 시청 범주의 고착화를 차단하는 신선도 지표입니다.
D-Index
평소 자주 즐기던 메인 풀 영역과 추천할 타겟 장르 간의 '벡터 거리(이질성)'를 연산해 필터 버블 바깥으로 시야를 넓힙니다.
N-Index
겉보기엔 비유사하나 숨어있는 깊은 메타데이터 차원의 연관 계수를 추출해 진정한 발견의 우연성(Serendipity)을 견인합니다. (50% 배정)
H-Index
생성형 AI 복제 피드와 대조되는 인간 창작물의 불완전하지만 가치 있는 노고의 총량으로, 유저 집단지성 투표 시스템을 기반으로 산출됩니다.
Formula
SRS점수 = (신선도×0.2) + (다양성×0.3) + (참신함×0.5)
최종점수 = SRS점수 × (1 + α × H)
2학년 연구 모델(SRS 정규화 지수)을 베이스 삼아, 3학년의 핵심 기획안인 유저 알고리즘 제어 수치 알파(α)와 인간성(H) 가중치를 융합한 다단 메커니즘입니다.
