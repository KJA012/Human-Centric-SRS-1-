# Human-Centric SRS Web Platform

React 기반의 인간 중심 추천 알고리즘 웹사이트입니다.

## 기능

- 📱 반응형 디자인
- 🎮 인터랙티브 데모
- 📊 데이터 시각화
- 🎯 학술적 설명

## 설치

```bash
cd web
npm install
```

## 실행

```bash
npm start
```

브라우저에서 http://localhost:3000 을 열어 확인하세요.

## 빌드

```bash
npm run build
```

## 배포

GitHub Pages, Vercel, Netlify 등으로 배포 가능합니다.
